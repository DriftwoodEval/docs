## Cloud

The cloud is a distributed collection of servers that host software and files. Most websites you visit are in the cloud.

## Database

A database is an organized collection of data, like a complex and more computer-friendly spreadsheet.

## Running Locally

Programs that run locally run on _your_ computer, as opposed to the [cloud](/glossary#cloud).
